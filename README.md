The goal of this task is to study unemployment trends using real data and understand how unemployment changed over time, especially during the Covid-19 period, using Python.
I do not build a prediction model here but is doing data analysis + visualization.

**Road-Map of the task** : 
1. Understand unemployment dataset and its columns
2. Clean data (date formatting, missing values, column clarity)
3. Analyze unemployment trends over time
4. Compare unemployment across regions
5. Identify spikes and drops in unemployment rates
6. Study the impact of Covid-19 (pre, during, post period)
7. Detect long-term and seasonal patterns
8. Use visualizations for trend interpretation
9. Demonstrate data analysis and visualization skills
